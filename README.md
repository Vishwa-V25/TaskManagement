# 📝 Task Management App (Frontend)

A modern and responsive **Task Management Web Application** built using **React.js and CSS**, designed to help users efficiently create, manage, and track their daily tasks in a clean and intuitive interface.

---

## 🚀 Live Demo
task-management-peach-delta.vercel.app
---

## 📌 Features

- ➕ Add new tasks easily
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed / pending
- 🔍 Filter tasks based on status
- 💾 Persistent UI state (if implemented using local storage)
- 📱 Fully responsive design (mobile + desktop)
- 🎨 Clean and modern UI using CSS

---

## 🛠️ Tech Stack

- ⚛️ React.js (Frontend Library)
- 🎨 CSS3 (Styling)
- 🧠 JavaScript (Logic handling)
- 📦 Node Package Manager (NPM)
